# LifeLink - マインドマップ（Markdown形式）

## 中心概念: LifeLink プラットフォーム

### 1. プロジェクト概要
- 1.1 ビジョン・ミッション
  - デジタル時代の記憶継承
  - 経済的・物理的負担の軽減
  - 世代を超えた絆の形成
- 1.2 解決する課題
  - 従来のお墓の高コスト（平均200万円）
  - デジタル遺産の散逸
  - 世代間断絶
  - 終活の複雑さ
- 1.3 ターゲット市場
  - プライマリ: 40-60代（4,300万人）
  - セカンダリ: 30-40代（2,800万人）
  - ターシャリ: 60代以上（4,000万人）

### 2. 市場分析
- 2.1 市場規模
  - グローバル: 39兆円（2025年）→ 117兆円（2034年）
  - CAGR: 12.97%
  - 日本: 257億円（2025年、前年比109.7%成長）
- 2.2 競合
  - Eterneva（米国）: 物理商品、高価格
  - Trust & Will（米国）: 遺言作成、日本未対応
  - Legacy.com（米国）: 追悼ページ、機能限定
  - 国内競合: 小規模、特化型
- 2.3 競合優位性
  - 日本市場特化
  - 総合的デジタル遺産管理
  - 家系図・DNA保存統合
  - VR/AR供養空間
  - 適正価格設定

### 3. 主要機能
- 3.1 デジタルメモリアルスペース
  - 3.1.1 プロフィールページ
    - 基本情報表示・編集
    - プロフィール写真（最大5MB）
    - 自己紹介（最大1,000文字）
    - 人生のマイルストーン
    - 統計情報
  - 3.1.2 メディアギャラリー
    - 画像: JPEG, PNG, WebP, GIF（10MB以下）
    - 動画: MP4, MOV, AVI（500MB以下）
    - 音声: MP3, WAV, OGG（50MB以下）
    - ドラッグ&ドロップアップロード
    - サムネイル自動生成
    - タグ付け・検索
  - 3.1.3 未来へのメッセージカプセル
    - テキスト・ビデオ・音声メッセージ
    - 配信条件設定
      - 日付指定
      - 年齢指定
      - イベント指定
      - 死後配信
    - 通知方法: メール・アプリ内・SMS
  - 3.1.4 人生タイムライン
    - イベント種別: 誕生、卒業、就職、結婚等
    - 写真添付（最大10枚）
    - 表示形式: 縦型・横型・カレンダー・年表

- 3.2 家系図機能
  - 3.2.1 家系図作成・編集
    - 対応関係: 親子、配偶者、兄弟姉妹、祖父母-孫
    - インタラクティブ表示
    - ドラッグ&ドロップ編集
    - PDF/画像エクスポート
    - 家族招待機能
  - 3.2.2 DNA保存サービス（Phase 2）
    - DNA採取キット郵送
    - 専門機関保管
    - 遺伝子情報デジタル記録
    - 世代間変化追跡
    - 料金: 初回¥50,000、年間¥5,000

- 3.3 バーチャル供養システム
  - 3.3.1 デジタル墓地（3D）
    - 環境設定: 和風・洋風・モダン
    - 背景: 森・海・山・都市
    - 時間帯・天候設定
    - 墓石カスタマイズ
    - QRコード埋め込み
    - 技術: Three.js / WebGL
  - 3.3.2 オンライン法要
    - Zoom法要（最大50名）
    - 提携寺院接続
    - 録画・アーカイブ
    - プラン: ¥30,000〜¥50,000
    - 手数料: 20%
  - 3.3.3 お参り・お供え
    - デジタルお供え物: お花、お線香、ろうそく
    - 価格: ¥50〜¥300
    - メッセージ書き込み
    - 訪問記録
  - 3.3.4 QRコード連携
    - 墓石プレート: ¥30,000〜¥50,000
    - スキャンでアクセス
    - AR機能（Phase 2）

- 3.4 アクセス権限管理
  - 5段階権限レベル
    - レベル1: 本人のみ
    - レベル2: 家族
    - レベル3: 親戚・親しい友人
    - レベル4: 知人
    - レベル5: 一般公開
  - コンテンツ別設定
  - アクセスログ記録
  - 不正アクセス検知

- 3.5 AI支援機能（Phase 2-3）
  - 3.5.1 AIライティング支援
    - 自分史執筆サポート
    - メッセージ文章化
    - 多言語翻訳
    - モデル: OpenAI GPT-4
  - 3.5.2 写真・動画編集
    - 顔認識・分類
    - 重複削除
    - 画質向上（AI超解像）
    - 自動タグ付け
  - 3.5.3 AIアバター・音声合成（Phase 3）
    - デジタルツイン作成
    - 音声クローニング
    - 簡単な対話応答
    - 技術: ElevenLabs, D-ID

### 4. 技術アーキテクチャ
- 4.1 フロントエンド
  - Next.js 14 (App Router)
  - React 18 + TypeScript 5
  - Tailwind CSS + shadcn/ui
  - Radix UI Components
  - PWA対応
- 4.2 バックエンド
  - Node.js 18
  - NextAuth.js（認証）
  - Prisma ORM
  - PostgreSQL 15
  - Redis（キャッシュ）
- 4.3 インフラ
  - Vercel（ホスティング推奨）
  - AWS / Google Cloud Platform
  - Cloudflare（CDN）
  - AWS S3（ストレージ）
  - Supabase / Neon（DB）
- 4.4 AI/ML
  - OpenAI GPT-4 API
  - ElevenLabs（音声合成）
  - Stable Diffusion（画像生成）
  - AWS Rekognition（顔認識）

### 5. データベース設計
- 5.1 主要テーブル（11テーブル）
  - users: ユーザー情報
  - family_relationships: 家系図関係
  - media_contents: メディアファイル
  - message_capsules: 未来へのメッセージ
  - access_permissions: 権限管理
  - memorial_visits: 供養訪問記録
  - dna_records: DNA保存記録
  - timelines: タイムラインイベント
  - qr_codes: QRコード管理
  - accounts & sessions: NextAuth用
- 5.2 パフォーマンス最適化
  - インデックス戦略: B-tree, GIN
  - パーティショニング: 時間ベース
  - クエリ最適化: N+1問題対策
  - キャッシング: Redis活用

### 6. セキュリティ
- 6.1 認証・認可
  - パスワードポリシー: 8文字以上、複雑性要件
  - JWT: アクセストークン15分、リフレッシュ30日
  - 二段階認証（2FA）: TOTP、SMS
- 6.2 データ保護
  - 転送時暗号化: TLS 1.3
  - 保存時暗号化: AES-256-GCM
  - パスワードハッシュ: bcrypt（ラウンド10）
- 6.3 脆弱性対策
  - OWASP Top 10対策
  - セキュリティヘッダー設定
  - 定期的セキュリティ監査
- 6.4 監査とコンプライアンス
  - 監査ログ: 2年保存
  - 個人情報保護法準拠
  - GDPR対応
  - PCI DSS（決済）

### 7. UI/UX設計
- 7.1 デザインシステム
  - カラーパレット: Primary（紫）、Secondary（紫）
  - タイポグラフィ: Noto Sans JP
  - スペーシング: 8pxベース
- 7.2 レスポンシブデザイン
  - ブレークポイント: sm(640px), md(768px), lg(1024px), xl(1280px)
  - モバイルファースト設計
  - タッチ対応: 44px以上
- 7.3 アクセシビリティ
  - WCAG 2.1 AA準拠
  - キーボード操作対応
  - スクリーンリーダー対応
  - コントラスト比: 4.5:1以上
- 7.4 高齢者向け配慮
  - フォントサイズ: 最小16px（推奨18px）
  - ボタンサイズ: 最小44×44px
  - 高コントラスト
  - シニアモード提供

  - 競合参入: 先行者利得、独自機能
  - 市場受容遅れ: 教育的マーケティング
  - 資金ショート: 計画的資金調達

**マインドマップの使い方**

このMarkdown形式のマインドマップは、以下のツールで可視化できます：

1. **Markmap**（推奨）
   - https://markmap.js.org/
   - Markdownから自動的にインタラクティブなマインドマップを生成

2. **MindMup**
   - https://www.mindmup.com/
   - Markdown形式をインポート可能

3. **XMind**
   - Markdown形式をインポート
   - より高度な編集が可能

4. **VS Code拡張機能**
   - Markdown Preview Enhanced
   - Markdownをマインドマップとして表示

**インデントレベル**:
- #: レベル0（中心）
- ##: レベル1（主要カテゴリ）
- ###: レベル2（サブカテゴリ）
- -: レベル3（詳細項目）
- さらにインデント: レベル4以降
